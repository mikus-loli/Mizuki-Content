# Live2D API

Live2D 看板娘插件后端 API，支持 Cubism 2.x 和 Cubism 4.x 模型，提供完整的 Web 管理后台和 Docker 部署方案。

::github{repo="mikus-loli/live2d_api"}

**API 地址**: [https://live2d.mikus.ink](https://live2d.mikus.ink)
